# mcmrc  

## 목적  
``` 
 - Vuejs를 이용한 toy project 일환  
 - Vuejs에서 많이 사용되는 라이브러리 사용법 및 Vuejs 프로젝 구조 학습
 - 노출데이타는 api통신이 아닌 static data를 사용
 - css는 적용하지 않음..
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
